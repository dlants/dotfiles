# dotfiles

my dotfiles for vim, zsh, i3, etc
gratefully copied from https://github.com/dlants/dotfiles & customized on `mine` branch.

- managed via nix + home-manager (`flake.nix`, `nix/`, `setup.sh`)
- **active:** `nvim/`, `starship.toml`, `ghostty/`, git
- **installed but not default:** `fish/` (login shell is still zsh)
- **wired up, used ad-hoc:** `tmux.conf` + `scripts/` (ta, clipboard-sync, etc.)
- **configured but app not installed yet:** `hammerspoon/` (run brew install if you want it)
- setup & use guides: `docs/`
- AI/agent helpers: `magenta-skills/`; experimental/old bits: `archive/`
